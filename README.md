# 🎬 VidSnap AI (Video Generation System)

VidSnap AI is a web-based application that generates videos using AI-powered media processing.
It uses a simple and responsive interface built with Bootstrap for better user experience.

---

## 🚀 Features

* 🎥 AI-based video generation
* 🔊 Audio processing using ElevenLabs
* ⚡ Fast video processing using FFmpeg
* 🌐 Responsive UI using Bootstrap
* 📁 Efficient file handling and management

---

## 🛠️ Technologies Used

### Backend:

* Python
* Flask

### Frontend:

* HTML
* CSS
* Bootstrap

### Media Processing:

* FFmpeg
* ElevenLabs

### Utilities:

* os
* time
* uuid

### Output:

* MP4 Video Format
